# 🔀 Version Control with Git

Welcome to the **Git** section.

Git is the industry standard for Source Code Management (SCM). It enables collaboration, versioning, and is the core trigger for most CI/CD pipelines.

## 🎯 Learning Objectives
- Master basic Git workflow (add, commit, push, pull).
- Understand Branching and Merging strategies (GitFlow vs. Trunk-based).
- Learn advanced operations: Rebasing, Cherry-picking, and Stashing.
- Practice collaborative workflows via Pull Requests and Code Reviews.
