# 🐧 Linux & Shell Scripting

Welcome to the **Linux & Shell Scripting** section of the DevOps roadmap.

Linux is the foundational operating system for the vast majority of servers, cloud environments, and container runtimes. Mastering the Linux command line and automation via Shell scripting is the first step in any DevOps journey.

## 🎯 Learning Objectives
- Understand the Linux File System Hierarchy.
- Master core CLI tools for file manipulation, searching, and process management.
- Learn User and Group management and permissions (chmod, chown).
- Automate repetitive tasks using Bash scripting.
