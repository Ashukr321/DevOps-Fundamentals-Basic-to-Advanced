# 🔒 DevSecOps: Security in the Lifecycle

Welcome to the **DevSecOps** section.

DevSecOps is the practice of integrating security into every stage of the software development lifecycle. It shifts security "left," making it a shared responsibility rather than an afterthought.

## 🎯 Learning Objectives
- Learn about SAST (Static) and DAST (Dynamic) security testing.
- Implement Container Scanning for vulnerabilities (e.g., Trivy).
- Master Secrets Management using tools like HashiCorp Vault.
- Integrate automated security gates into CI/CD pipelines.
