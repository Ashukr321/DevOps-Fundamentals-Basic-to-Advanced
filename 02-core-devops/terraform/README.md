# 🏗️ Infrastructure as Code (IaC) with Terraform

Welcome to the **Terraform** section.

Infrastructure as Code (IaC) allows you to manage and provision infrastructure through machine-readable definition files, rather than manual hardware configuration or interactive configuration tools.

## 🎯 Learning Objectives
- Understand Declarative vs. Imperative infrastructure management.
- Master Terraform syntax (HCL) and core commands (init, plan, apply, destroy).
- Learn about State management and remote backends.
- Build reusable modules for consistent infrastructure deployment.
